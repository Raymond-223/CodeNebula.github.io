算法学习笔记

左侧目录会自动同步 algorithm 文件夹中的所有笔记（.md/.html 格式），上传/删除/重命名笔记后，目录将在 1-3 分钟内自动更新，无需手动修改代码。

如何添加笔记？

1. 本地新建笔记文件（推荐 Markdown 格式：.md，支持代码高亮、公式、列表）
2. 上传文件到 GitHub 仓库的 algorithm 文件夹（直接上传，无需分类子文件夹）
3. 等待 1-3 分钟（GitHub 自动构建）
4. 左侧目录将自动显示新笔记，点击即可访问
支持格式：.md / .html 自动同步目录

注意事项

✅ 笔记文件名不要含空格和特殊字符（示例：「排序算法.md」✅，「排序 算法.md」❌）
✅ Markdown 笔记支持代码高亮（用 ```语言名 包裹代码，如 ```cpp ```python）
✅ 删除笔记后，目录会自动移除对应选项
✅ 笔记中若有图片，建议放在 algorithm/images 文件夹，用相对路径引用（如 ![图1](images/xxx.png)）
区分大小写 实时同步更新
